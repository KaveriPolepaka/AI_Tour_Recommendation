# Conversational AI Travel Recommendation System

## Overview

This project is a conversational AI-based travel recommendation system that provides personalized destination suggestions based on user preferences such as budget, mood, and travel interests.

The system uses Python, FastAPI, and recommendation logic to generate intelligent travel recommendations including destinations, estimated costs, and travel categories.

---

## Features

* Conversational travel recommendation system
* Personalized destination suggestions
* Budget-based filtering
* Travel category recommendations
* FastAPI backend integration
* CSV dataset processing
* Intelligent recommendation generation

---

## Challenges Faced

One major challenge was handling inconsistent response formats returned by different recommendation modules. This caused runtime issues during recommendation generation. The issue was solved by standardizing outputs into a consistent structure.
